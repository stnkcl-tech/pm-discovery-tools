# User Journey Map

| Stage | User Action | Touchpoint | Pain Point | Emotion | Opportunity |
|-------|-------------|------------|------------|---------|-------------|
| **Awareness** | Realizes need for connection | Internal survey / 1:1 | No clear signal | Anxious | Proactive pulse checks |
| **Consideration** | Explores options | Slack, HR tools | Overwhelming choice | Frustrated | Curated recommendations |
| **Decision** | Selects a solution | Team vote / manager pick | Budget constraints | Cautious | Free tier / pilot program |
| **Execution** | Uses tool/process | Daily workflows | Adoption friction | Hopeful | Seamless integration |
| **Completion** | Feels more connected | Self-reflection / feedback | Hard to measure | Satisfied | Clear progress indicators |
